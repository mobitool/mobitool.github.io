## mobi 格式
.mobi 是 Kindle 阅读器支持的一种文件格式。

现可以通过邮件方式，将以下类型文件，直接发送到 kindle 邮箱，用户无需在进行转换。

### 支持文件的类型
+ Microsoft Word (.DOC, .DOCX)
+ HTML (.HTML, .HTM)
+ RTF (.RTF)
+ JPEG (.JPEG, .JPG)
+ Kindle Format (.MOBI, .AZW)
+ GIF (.GIF)
+ PNG (.PNG)
+ BMP (.BMP)
+ PDF (.PDF)
